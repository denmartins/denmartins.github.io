Perceptron

  • Primeira mquina capaz de aprender a partir de dados.
  • Consiste em um conjunto de pesos $\mathbf{w}\in\mathbb{R}^{m}$ associados s $m$ caractersticas (features) de entrada $\mathbf{x}$
  • Computa uma sada $z=\mathbf{w}^{T}\mathbf{x}+b$, onde $b$ um vis escalar.
  • Aplica uma funo de ativao $\sigma$ que transforma $z$.
$$ \sigma(z) = \begin{cases} 0 & z\leq 0 \\ 1 & z \gt 0 \end{cases} $$		 Viso Geral do Perceptron. Image Source: Sebastian Raschka.

Perceptron Learning Rule

  • Dado um exemplo $(\mathbf{x},y)$, onde $y\in\{0,1\}$ o rtulo verdadeiro, calculase a predio $\hat{y}=f(z)$.
  • Se a predio estiver incorreta ($e=y-\hat{y}\neq 0$), os pesos e vis so ajustados na direo que reduz o erro.
  • $\mathbf{w}\leftarrow\mathbf{w}+\eta\,e\,\mathbf{x}$ e $b\leftarrow b+\eta\,e$, onde $\eta>0$ a taxa de aprendizado.

class Perceptron():
    def __init__(self, num_features):
        self.num_features = num_features
        self.weights = torch.zeros(
            num_features, 1, dtype=torch.float32, device=device)
        self.bias  = torch.zeros(1, dtype=torch.float32, device=device)
        self.ones  = torch.ones(1, device=device) 
        self.zeros = torch.zeros(1, device=device)

    def forward(self, x):
        linear = torch.mm(x, self.weights) + self.bias
        predictions = torch.where(linear > 0., self.ones, self.zeros)
        return predictions
        
    def backward(self, x, y):  
        predictions = self.forward(x)
        errors = y - predictions
        return errors
        
    def train(self, x, y, epochs):
        for _ in range(epochs):
            for i in range(y.shape[0]):
                errors = self.backward(
                    x[i].reshape(1, self.num_features), y[i]).reshape(-1)
                self.weights += (errors * x[i]).reshape(self.num_features, 1)
                self.bias += errors
                
    def evaluate(self, x, y):
        predictions = self.forward(x).reshape(-1)
        accuracy = torch.sum(predictions == y).float() / y.shape[0]
        return accuracy

Adaline: Adaptive Linear Neuron

  • Introduzido por Widrow e Hoff em 1960.
  • Funo de ativao linear $\sigma$.
  • Funo de custo o MSE $\mathcal{L}(w,b) = \frac{1}{m} \sum^m_{i=1} (y^i - \sigma(z^i))^2$.
  • Perceptron aprimorado: treinamento busca minimizar o MSE em vez de simplesmente corrigir classificaes incorretas.
  • $\sigma$ linear permite o gradiente do erro seja calculado de forma contnua.
  • $\sigma$ convexa, permitindo usar o algoritmo de otimizao gradiente descente.
 Adaline. Image Source: Sebastian Raschka.

Gradiente Descendente

Atualizao de parmetros do modelo utilizando informao do gradiente de uma funo de custo (loss) $\mathcal{L}(w,b)$: $\nabla(\mathcal{L}(w,b))$.

Ou seja, dar um passo na direo oposta do gradiente. $$\Delta w = - \eta \times \nabla_w(\mathcal{L}(w,b))$$ $$\Delta b = - \eta \times \nabla_b(\mathcal{L}(w,b))$$

  • $\eta$: learning rate (tamanho do passo)

Gradiente da funo de custo com respeito a $w_j$ e a $b$:

$$\frac{\partial \mathcal{L}}{\partial w_j} = - \frac{1}{m} \sum_i (y^i - \sigma(z^i))x^i_j$$$$\frac{\partial \mathcal{L}}{\partial b} = - \frac{1}{m} \sum_i (y^i - \sigma(z^i))$$

Gradiente Descendente (cont.)

Podemos escrever a atualizao de weights e bias como:

  • $\Delta w_j = - \eta (\frac{\partial \mathcal{L}}{\partial w_j})$
  • $\Delta b = - \eta (\frac{\partial \mathcal{L}}{\partial b)}$

Usamos um algoritmo para computar gradientes com base no conjunto completo de dados de treinamento e atualizar os parmetros do modelo. Essa atualizao se d realizando um pequeno passo na direo oposta do gradiente da loss $\Delta \mathcal{L}(w,b)$.

MLP: Treinamento

O procedimento de treinamento do MLP pode ser resumido em trs etapas simples:

  1. A partir da camada de entrada, propagamos adiante ("feedforward") os padres dos dados de treinamento atravs da rede para gerar uma sada.
  2. Com base na sada da rede, calculamos a perda (loss) que queremos minimizar usando uma funo de perda que descreveremos mais adiante.
  3. Propagamos a loss para trs (backpropagation), determinamos sua derivada em relao a cada peso e vis da rede, e atualizamos o modelo.

Por fim, aps repetirmos essas trs etapas por mltiplas pocas e aprendermos os parmetros de peso e vis do MLP, usamos a propagao adiante para calcular a sada da rede e aplicamos uma funo de limiar (threshold) para obter as classes previstas.

MLP: Feed-forward

Vamos analisar passo a passo a propagao adiante para gerar uma sada a partir dos padres presentes nos dados de treinamento. Como cada unidade da camada oculta est conectada a todas as unidades das camadas de entrada, primeiro calculamos a unidade de ativao da camada oculta $a_1^{(h)}$ da seguinte forma:

$$ z_1^{(h)} = x_1^{(\text{in})}w_{1,1}^{(h)} + x_2^{(\text{in})}w_{1,2}^{(h)} + \ldots + x_m^{(\text{in})}w_{1,m}^{(h)} $$$$ a_1^{(h)} = \sigma(z_1^{(h)}) $$

Aqui, $z_1^{(h)}$ a entrada lquida (net input) e $\sigma(.)$ a funo de ativao, que deve ser diferencivel para permitir o aprendizado dos pesos que conectam os neurnios por meio de uma abordagem baseada em gradiente. Para resolver problemas complexos, como classificao de imagens, precisamos de funes de ativao no lineares no nosso modelo MLP; um exemplo comum a funo sigmoide (logstica):

$$ \sigma(z) = \frac{1}{1 + e^{-z}} $$

A funo sigmoide uma curva em forma de S que mapeia a entrada lquida $z$ para uma distribuio logstica no intervalo de 0 a 1, cruzando o eixo $y$ quando $z = 0$.

MLP: Feed-Forward (cont.)

Escrevemos a ativao em uma forma mais compacta e vetorizada (para evitar loops): $$ z^{(h)} = x^{(\text{in})}W^{(h)T} + b^{(h)} $$

$$ a^{(h)} = \sigma(z^{(h)}) $$

Aqui, $x^{(\text{in})}$ o nosso vetor de caractersticas de dimenso $1\times m$.
$W^{(h)}$ uma matriz de pesos de dimenso $d\times m$, onde $d$ representa o nmero de unidades na camada oculta; consequentemente, a matriz transposta $W^{(h)T}$ tem dimenso $m\times d$.
O vetor de vis $b^{(h)}$ contm $d$ unidades de bias (uma para cada n oculto).

Aps a multiplicao matricialvetorial, obtemos o vetor de entrada lquida $\;z^{(h)}\;$ de dimenso $1\times d$, que ser usado para calcular a ativao $a^{(h)} \in \mathbb{R}^{1\times d}$.

MLP: Feed-Forward (cont.)

Podemos generalizar este clculo para os $n$ exemplos do conjunto de treinamento:

$$ Z^{(h)} = X^{(\text{in})}W^{(h)T} + b^{(h)} $$

Neste caso, $X^{(\text{in})}$ passa a ser uma matriz $n\times m$; a multiplicao matricial resulta em uma matriz de entrada lquida $\;Z^{(h)}\;$ de dimenso $n\times d$. Por fim, aplicamos a funo de ativao $\sigma(\cdot)$ a cada elemento da matriz de entrada lquida para obter a matriz de ativao $n\times d$ na camada seguinte (aqui, a camada de sada):

$$ A^{(h)} = \sigma(Z^{(h)}) $$

De maneira anloga, podemos escrever a ativao da camada de sada em forma vetorizada para mltiplos exemplos:

$$ Z^{(\text{out})} = A^{(h)}W^{(\text{out})T} + b^{(\text{out})} $$

Aqui multiplicamos a transposta da matriz $t\times d$ $W^{(\text{out})}$ (onde $t$ o nmero de unidades de sada) pela matriz $n\times d$ $A^{(h)}$, e somamos o vetor de vis de dimenso $t$, $b^{(\text{out})}$, obtendo a matriz $Z^{(\text{out})}$ de dimenso $n\times t$. (As linhas desta matriz representam as sadas para cada exemplo.)

Por fim, aplicamos a funo sigmoide para obter o valor contnuo da sada do nosso modelo:

$$ A^{(\text{out})} = \sigma(Z^{(\text{out})}) $$

import torch.nn as nn
import torch.nn.functional as F

class MLPNet(nn.Module):
    def __init__(self):
        super(MLPNet, self).__init__()
        self.flatten = nn.Flatten()
        # input layer
        self.fc1 = nn.Linear(28 * 28, 64)
        # linear layer (n_hidden -> hidden_2)
        self.fc2 = nn.Linear(64, 32)
        # linear layer (n_hidden -> 10)
        self.fc3 = nn.Linear(32, 10)
        
    def forward(self, x):
        x = self.flatten(x)
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

Funo de Custo/Loss para Classificao

  • CrossEntropy Loss: $\mathcal{L}(\mathbf{y},\hat{\mathbf{y}})= -\sum_{k=1}^{K} y_k\,\log(\hat y_k)$
  • Intuio
    • Mede a divergncia entre o vetor de rtulos reais $y$ (ou $y_k$) e a distribuio predita $\hat{y}$.
    • Penaliza fortemente previses que atribuem baixa probabilidade ao verdadeiro rtulo.
  • Propriedades importantes
    • Nonegatividade: $\mathcal{L} \ge 0$; zero apenas quando $\hat y = y$.
    • Derivada simples: facilita a implementao do algoritmo de backpropagation.
  • Quando usar: Binria ou multiclasse quando as sadas so interpretadas como probabilidades (softmax ou sigmoid).

def get_predictions(model, iterator, device):
    model.eval()
    images, labels, probs = [], [], []
    with torch.no_grad():
        for (x, y) in iterator:
            x = x.view(-1, 28*28).to(device)
            y = y.to(device)
            
            y_pred = model(x)
            y_prob = F.softmax(y_pred, dim=-1)
        
            images.append(x.cpu())
            labels.append(y.cpu())
            probs.append(y_prob.cpu())
    images = torch.cat(images, dim=0)
    labels = torch.cat(labels, dim=0)
    probs = torch.cat(probs, dim=0)
    
    return images, labels, probs

Funo de Ativao na Camada de Sada

  • Camada de Sada com Softmax: Cabea de Classificao (Classification Head)
  • Propsito principal: Converter vetores de logits (sada da ltima camada escondida) $z \in \mathbb{R}^{K}$ em uma distribuio de probabilidade sobre $K$ classes.
  • Definio matemtica: $\sigma(z)_k = \frac{\exp(z_k)}{\sum_{j=1}^{K} \exp(z_j)}, \qquad k=1,\dots ,K$
  • Cada componente fica no intervalo $(0,1)$ e a soma totaliza $1$.
  • Derivada simples: $ \frac{\partial \sigma(z)_i}{\partial z_j} = \sigma(z)_i (\delta_{ij} - \sigma(z)_j)$, onde $\delta_{ij}$ a funo indicadora.
  • Junto com a loss CrossEntropy, forma o padro ouro Softmax + CrossEntropy.
Funo Softmax para classificao. Fonte: Abhisek Jana.

Resumo

  • Perceptron: Modelo linear + funo de ativao (step/sigmoid).
  • Algoritmo de aprendizagem: ajuste dos pesos com base no erro.
  • MultiLayer Perceptron (MLP): Estrutura em camadas: entrada mltiplas camadas ocultas sada.
  • Ativaes nolineares (ReLU, tanh, sigmoid) permitem modelar funes complexas.
  • Treinamento de Redes Neurais
    • Funo de perda: CrossEntropy (classificao) ou MSE (regresso).
    • Otimizadores: SGD, Adam, RMSProp; ajustes de taxa de aprendizado.
    • Backpropagation: clculo eficiente de gradientes via cadeia da regra (prxima aula).
  • Exerccio: Experimente diferentes arquiteturas (nmero de camadas, neurnios).
